## CSS修正ポリシー（今後の修正共通）
- 追加上書きだけで済ませず、対象セクションのCSS責務を同時に整理する
- 同じ要素への重複ルールはできる限り集約し、宣言の重なりを減らす
- セクション単位で関連スタイルをまとめ、読みやすい順序を維持する
- 古くて効いていないルールは削除する
- `!important` は必要最小限に限定する
- 見た目（UI）を変えずに整理することを最優先にする
- 仕上げ時に DevTools で最終的に適用される勝ちルールを確認する
